# ESTV Brand Guidelines

## Logo Files

| File | Description | Usage |
|------|-------------|-------|
| `logo-icon.png` | Icon logo | Favicon, app icon, small displays |
| `logo-banner.png` | ESTV Labs banner | Headers, documentation |
| `token-icon.png` | Token symbol | Token listings, exchanges |

## Brand Colors

| Color | Hex | Usage |
|-------|-----|-------|
| Primary Green | `#7CFC00` | Logo, CTAs, highlights |
| Dark Background | `#1A1A2E` | Backgrounds |
| White | `#FFFFFF` | Text on dark |

## Typography

- **Headings**: Bold, modern sans-serif
- **Body**: Clean, readable sans-serif

## Usage Guidelines

1. Maintain clear space around logo
2. Do not distort or rotate logo
3. Use approved color variations only
4. Minimum size: 32px height
