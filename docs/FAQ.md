# ESTV Token FAQ

## General Questions

### What is ESTV?
ESTV is a blockchain-powered esports media platform that enables fans to engage with content through Watch-to-Earn (W2E) model and Proof of Engagement (PoE) algorithm.

### What blockchain is ESTV built on?
ESTV is built on Solana blockchain for high throughput and low transaction costs.
