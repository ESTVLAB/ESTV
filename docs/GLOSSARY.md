# ESTV Glossary

## Blockchain Terms

### SPL Token
Solana Program Library Token - the token standard on Solana blockchain, similar to ERC-20 on Ethereum.

### Merkle Tree
A data structure used to efficiently verify data integrity. ESTV uses Merkle trees to batch reward claims.

### Multi-sig
Multi-signature wallet requiring multiple approvals for transactions, used for security.

### TPS
Transactions Per Second - Solana supports high TPS for scalable operations.

## ESTV Platform Terms

### W2E (Watch-to-Earn)
ESTV's model where users can earn rewards by watching and engaging with content.

### PoE (Proof of Engagement)
Algorithm that verifies genuine user engagement to prevent bot activity and reward real participation.

### ESTV Super App
Integrated mobile application for streaming, gaming, commerce, and wallet functionality.

### Ecosystem Rewards
Tokens allocated for rewarding user participation and platform activities.

## Token Economics

### TGE (Token Generation Event)
The initial creation and distribution of tokens.

### Vesting
Gradual release of tokens over time according to a schedule.

### Cliff
Lock-up period before any tokens begin vesting.

### Treasury
Reserved tokens managed by the project for operational purposes.

## Technical Architecture

### Anchor Framework
Solana development framework for building secure smart contracts.

### Sealevel
Solana's parallel transaction processing runtime.

### Claim State
On-chain record tracking whether a user has claimed their rewards.

## Platform Features

### NFT Marketplace
Platform feature for trading digital collectibles and in-game items.

### Physical Centers
ESTV's offline esports arenas in US, Korea, and Middle East.

### OTT Distribution
Over-The-Top content delivery through streaming platforms.
